This document provides a representation of the MQTT protocol using the Web Ontology Language (OWL). It defines a collection of OWL classes and properties that represent the MQTT vocabulary as defined by the MQTT specification version 3.1.1 [[MQTT-311]] and version 5 [[MQTT-5]]. These OWL terms can be used to record MQTT messages in OWL.

This specification is not intended to reach the W3C recommendation status. Instead, it is intended to become a W3C Group Note companioning the W3C Thing Description recommendation.

Namespaces

The namespace for MQTT vocabulary terms is http://www.w3.org/ns/mqtt#

The suggested prefix for the MQTT vocabulary namespace is mqtt

The Turtle version of the MQTT vocabulary is available at http://www.w3.org/ns/mqtt/mqtt.ttl
