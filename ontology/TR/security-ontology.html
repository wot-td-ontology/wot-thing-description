This document provides a representation of the Security vocabulary used in the WoT Thing Description.

This specification is not intended to reach the W3C recommendation status. Instead, it is intended to become a W3C Group Note companioning the W3C Thing Description recommendation.

Namespaces

The namespace for the Security ontology terms is http://www.w3.org/ns/wot-security#

The suggested prefix for the Security ontology namespace is wotec

The Turtle version of the Security ontology is available at http://www.w3.org/ns/wot-security/wot-security.ttl
