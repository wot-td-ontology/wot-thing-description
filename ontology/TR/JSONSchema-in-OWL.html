This document provides a representation of the JSON Schema vocabulary in the Web Ontology Language (OWL), to allow tools to validate the JSON messages that are exchanged between Web agents. The RDF terms defined in this document represent the core JSON Schema specification defined by [[JSON-SCHEMA-CORE]].

This specification is not intended to reach the W3C recommendation status. Instead, it is intended to become a W3C Group Note companioning the W3C Thing Description recommendation.

Namespaces

The namespace for JSON Schema vocabulary terms is http://www.w3.org/ns/json-schema#

The suggested prefix for the JSON Schema vocabulary namespace is jsonschema

The Turtle version of the JSON Schema vocabulary is available at http://www.w3.org/ns/json-schema/json-schema.ttl

jsonschema:IntegerSchema

IRI: http://www.w3.org/ns/json-schema#IntegerSchema

a OWL Class

$label - A JSON integer value specification, that is, numbers without a fractional part ("type": "integer").

Sub class of jsonschema:DataSchema
Equivalent to jsonschema:type HAS VALUE "integer"
Restrictions jsonschema:minimum ONLY xsd:integer
jsonschema:maximum ONLY xsd:integer
jsonschema:NumberSchema

IRI: http://www.w3.org/ns/json-schema#NumberSchema

$label - A JSON number value specification ("type": "number").

Restrictions jsonschema:minimum ONLY union of xsd:double and xsd:integer
