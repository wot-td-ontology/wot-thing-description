This document provides a representation of the CoAP protocol using the Web Ontology Language (OWL). It defines a collection of OWL classes and properties that represent the CoAP vocabulary as defined by the CoAP specification [[CoAP]]. These OWL terms can be used to record CoAP messages in OWL.

This specification is not intended to reach the W3C recommendation status. Instead, it is intended to become a W3C Group Note companioning the W3C Thing Description recommendation.

Namespaces

The namespace for CoAP vocabulary terms is http://www.w3.org/ns/coap#

The suggested prefix for the CoAP vocabulary namespace is coap

The Turtle version of the CoAP vocabulary is available at http://www.w3.org/ns/coap/coap.ttl
